# Claro

Landing page and legal docs for Claro — a bilingual AI assistant for independent home service operators (house cleaners, handymen). Handles SMS booking, missed call auto-text, and customer follow-up in English and Spanish.

Hosted via GitHub Pages at [kblabs20.github.io/chiefofstaff-site](https://kblabs20.github.io/chiefofstaff-site/).

Built by KBLabs20. Early stage — not open source.
